# Basic Network Sniffer

Project Overview

The Basic Network Sniffer is a Python program that captures and analyzes network traffic packets in real time. It helps understand how data flows across a network and how different protocols operate.

The program captures packets and displays useful information such as source IP address, destination IP address, protocol type, and payload data.

## Objectives;

Capture network packets from a local network.

Analyze packet structure and protocol information.

Understand how network communication works.

Learn basic concepts of packet sniffing and network monitoring.

### Tools and Technologies;

Programming Language: Python

Library: Scapy

Packet Capture Driver: Npcap (for Windows)

#### Features;

Captures live network packets

Displays source and destination IP addresses

Identifies protocol types (TCP, UDP, ICMP)

Shows packet payload information

Helps analyze network traffic behavior

### Learning Outcomes

Understanding packet structure

Basic network traffic analysis

Familiarity with network protocols

Practical exposure to packet sniffing tools


## Conclusion

The Basic Network Sniffer project demonstrates how network packets can be captured and analyzed using Python. It provides a practical understanding of networking protocols and helps build foundational skills in network security and monitoring.
